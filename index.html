esensi

n. essence, an awesome Laravel boilerplate application

esensi/build

A package that uses Gulp tasks to pre-process CSS, JS, fonts and image assets into optimized builds that modern web projects need. It's a great front-end package for Laravel applications.

esensi/core

A package that provides the foundational components of Esensi, a platform built on Laravel. This package provides core patterns and commonly extended base classes to all other packages of the Esensi platform.

esensi/loaders

Laravel 4 used the same loader for translations, view services, and config files. In Laravel 5, the loader stopped supporting namespaced configs. This package fixes that up, and also adds namespaced alias files for better package development!

esensi/model

A package that extends Laravel's Eloquent ORM with traits suitable for validating, purging, hashing, encrypting, and soft-deleting models with ease. It also supports short-hand for model relationships.
